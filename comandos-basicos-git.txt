Principais comandos git

1- Configurando nome do usuário
git config --global user.name nome-do-seu-usuário

2- Configurando e-mail do usuário
git config --global user.email e-mail-do-seu-usuário

3- Verificar o usuário configurando git config user.name

4- Verificar o email configurando git config user.email

5- Iniciar um repositório
git init

6- Adicionar os arquivos para o envio ao Github
git add .

7- Adicionar mensagem sobre os arquivos para o envio ao Github
git commit -m "arquivos iniciais"

OBS: Depois de confirmado o envio do repositório local para o remoto,
toda alteração posterio deverá ser enviada ao Github com os comandos a seguir:

git add.
git commit -m "Nova mensagem"
got push
